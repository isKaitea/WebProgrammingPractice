定位
1. 沒有 position 就不能使用 z-index（無法堆疊）。
2. relative：保留位置；可當 absolute 子元素的定位基準。
3. absolute：脫離排版；依照最近有 position 的父元素定位。
4. 常用組合：父 relative + 子 absolute。
5. z-index：定位後才有效；數字越大越上層。

第七堂課：JavaScript — 函數 / 變數常數 / 物件 / 陣列

HTML
  • <script> 內可撰寫 JavaScript 程式。
  • 使用 id 搭配 document.getElementById() 取得元素。
  • <button onclick="functionName()"> 可直接在 HTML 呼叫函數。
  • input 元件可用 .value 取得輸入內容。
JavaScript
  • 函數（Function）：可封裝程式區塊並重複呼叫。
    • function showMessage(){ ... }
  • 判斷式（if / else）：根據條件決定要執行哪段程式。
    •                                     if (條件) {
                                        // true 時執行
                                    } else {
                                        // false 時執行
                                    }
  • 多條件判斷（else if）：依序檢查多個條件，最先符合的會被執行。
    •                                     if (條件1) {
                                    } else if (條件2) {
                                    } else if (條件3) {
                                    } else {
                                    }
  • 變數：
    • var：可重複宣告（不推薦）。
    • let：區塊作用域，可重新賦值。
    • const：常數，不可重新賦值。
  • 物件（Object）：由屬性 + 方法組成。
    • var car = {model:"Toyota", start:function(){...}};
  • 陣列（Array）：儲存多個資料，可用索引讀取。
    • var cars = ["A","B","C"];
  • 陣列常用方法：
    • .sort()：排序
    • .reverse()：反轉
    • .push() / .pop()：尾端新增 / 移除
    • .unshift() / .shift()：開頭新增 / 移除
    • .join()：以字串輸出陣列內容
    • .indexOf()：查找指定元素位置
  • 取值與呈現：
    • document.getElementById("msg").innerText = ...

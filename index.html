第七堂課：JavaScript — 函數 / 變數常數 / 物件 / 陣列

HTML
  • <script> 內可撰寫 JavaScript 程式。
  • 使用 id 搭配 document.getElementById() 取得元素。
  • <button onclick="functionName()"> 可直接在 HTML 呼叫函數。
  • input 元件可用 .value 取得輸入內容。
JavaScript
  • 函數（Function）：可封裝程式區塊並重複呼叫。
    • function showMessage(){ ... }
  • 判斷式（if / else）：根據條件決定要執行哪段程式。
    • if (條件) {
    // true 時執行
} else {
    // false 時執行
}
  • 多條件判斷（else if）：依序檢查多個條件，最先符合的會被執行。
    • if (條件1) {
} else if (條件2) {
} else if (條件3) {
} else {
}
  • 變數：
    • var：可重複宣告（不推薦）。
    • let：區塊作用域，可重新賦值。
    • const：常數，不可重新賦值。
  • 物件（Object）：由屬性 + 方法組成。
    • var car = {model:"Toyota", start:function(){...}};
  • 陣列（Array）：儲存多個資料，可用索引讀取。
    • var cars = ["A","B","C"];
  • 陣列常用方法：
    • .sort()：排序
    • .reverse()：反轉
    • .push() / .pop()：尾端新增 / 移除
    • .unshift() / .shift()：開頭新增 / 移除
    • .join()：以字串輸出陣列內容
    • .indexOf()：查找指定元素位置
  • 取值與呈現：
    • document.getElementById("msg").innerText = ...

第八堂課：Switch & 迴圈

課堂練習 js01： 音樂和我分不開


課堂練習 js02： 關西賞櫻景點

HTML
  • <select> 搭配 onchange 屬性可偵測選項改變。
  • <option> 的 value 可傳入 switch 或判斷式。
  • <p> 可用 innerHTML 或 innerText 動態更新。
CSS 本堂課無新增 CSS 語法。
JavaScript
  • Switch 判斷式：適合多固定選項的情境。
    • switch (表達式) {
    case 值1:
        // 當表達式 === 值1
        break;
    case 值2:
        // 當表達式 === 值2
        break;
    default:
        // 以上都不符合時執行
}
  • 示範：星期判斷（搭配下拉選單 onchange）
  • for 迴圈：重複執行指定次數。
    • for (初始化; 條件; 步進值) {
    // 重複的程式碼
}
  • 示範：自動產生 120 年的年份選單（動態加入 option）
  • for...in：用於陣列或物件的索引迴圈。
    • for (索引 in 陣列) {
    // 透過 陣列[索引] 取得資料
}
  • while 迴圈：先判斷，再執行。
    • while (條件) {
    // 重複執行
}
  • do...while 迴圈：至少執行一次。
    • do {
    // 先執行一次
} while (條件);
  • break 與 continue
    • // break：直接跳出迴圈
for (let i = 0; i < 10; i++) {
    if (i === 5) break;
}

// continue：跳過本次迴圈，進入下一次
for (let i = 0; i < 10; i++) {
    if (i % 2 === 0) continue; // 偶數跳過
}
  • typeof：用來檢查變數型別。
    • typeof 123       // "number"
typeof "hello"   // "string"
typeof true      // "boolean"
typeof []        // "object"
typeof {}        // "object"
typeof undefined // "undefined"

第九堂課：AJAX 與 jQuery

課堂練習 web09： [09.html] jQuery 事件、AJAX 讀取 JSON、表單驗證

HTML
  • 使用 <script src="jquery.js"> 載入 jQuery。
  • <select> 搭配 onchange 可觸發選單事件。
  • <form> 若回傳 false 可阻止表單送出。
  • AJAX 通常使用一個容器如 <div id="news"> 來顯示資料。
JavaScript
  • AJAX 非同步讀取 JSON
    • $.ajax({
    url: 'news.json',
    dataType: 'json',
    success: function(data){
        $.each(data, function(i, item){
            $('#news').append(
                '<li>' + item.date + '-' + item.title + '</li>'
            );
        });
    },
    error: function(){
        console.log('Error fetching data');
    }
});
  • jQuery 基本語法：$(selector).action()
    • // ready（舊寫法）
$(document).ready(function(){
    // code
});

// ready（新寫法）
$(function(){
    // code
});
  • 事件：click() — 隱藏所有段落
    • $('button').click(function(){
    $('p').hide();
});
  • 指定 ID 選擇器
    • $('#p1').hide();
$('#p2').show();
  • focus / blur：輸入框事件
    • $('#txt1').focus(function(){
    $(this).css('background-color', 'lightblue');
});

$('#txt2').blur(function(){
    $(this).css('background-color', 'lightblue');
});
  • submit 事件：讀取表單資料 + 阻止送出
    • $('#form1').on('submit', function(){
    let username = $('#username').val();
    let usertel  = $('#usertel').val();
    alert('姓名:' + username + '\\n電話:' + usertel);
    return false;  // 阻止送出
});
  • 查詢功能：模仿 Google 搜尋

    • 使用 <form action="https://www.google.com/search">，並將搜尋字串放在 name="q"。

  • 註冊表單驗證
    • $('#form_register').on('submit', function(){
    if($('#name').val()==='' || $('#user').val()===''){
        $('#msg').text('請輸入所有欄位');
        return false;
    }
    if($('#password').val() !== $('#confirm_pw').val()){
        $('#msg').text('密碼不一致');
        return false;
    }
});
  • 常見錯誤修正：
    • $this → ❌（錯誤）
    • $(this) → ✔ 正確寫法

第十堂課：Bootstrap 5

課堂練習 web10： Bootstrap 範例網格排版

↑ 回到最上
HTML
  • 使用 CDN 引入 Bootstrap 5 CSS 與 JS

CSS
  • .container：固定寬度容器（隨斷點變化）
  • .container-fluid：滿版容器 100%
  • .img-fluid：圖片自動縮放以適應裝置
  • 文字大小：display-1 ～ display-6
  • 色彩類別：primary、secondary、success、danger、warning、info
  • 間距類別：m-、p-、mt-、mx-、py- 等等

  • (place) → 產生 placeholder 圖片（VS Code 外掛）
  • div.row → 產生 Bootstrap row
  • img.img-fluid → RWD 圖片
  • table.table.table-striped → 快速產生表格樣式

補充筆記

  • Bootstrap 一行共 12 欄（Grid System）
  • col-3 col-md-6：手機 4 格 → 平板 / PC 2 格
  • offset-sm-2：在小螢幕向右推移 2 欄
  • display-1 用於標題，字體最大
  • table-responsive-sm：表格在手機可左右滑動

第十一堂課：Bootstrap 元件與導航

課堂作業 web11： Bootstrap 導覽列、輪播與元件整合頁面

HTML
  • 導覽列：<nav class="navbar ..."> 建立上方固定導航（fixed-top）。
  • 麵包屑導航：<nav class="breadcrumb"> + .breadcrumb-item 表示目前位置。
  • 輪播圖：.carousel、.carousel-inner、.carousel-item 搭配指示點與左右控制箭頭。
  • 提示框：.alert 顯示訊息，alert-dismissible + 關閉按鈕可關閉。
  • 圖片區：<figure> + <figcaption> 搭配 .figure-img、.img-fluid。
  • 卡片元件：.card 包含 .card-img-top、.card-body、.card-title、.card-text。
  • 下拉選單：.dropdown、.dropdown-menu、.dropdown-item。
  • 按鈕與按鈕群組：.btn 搭配 .btn-group、.btn-group-vertical、checkbox / radio。
  • 徽章：.badge 搭配 .position-relative、.position-absolute 做小紅點效果。
  • 分頁：.pagination + .page-item、.page-link 建立頁碼列。
  • 列表元件：.list-group、.list-group-item 及 .list-group-numbered。
  • 摺疊面板：.collapse 配合 data-bs-toggle="collapse" 控制顯示 / 收合。
  • 手風琴（Accordion）：.accordion、.accordion-item、.accordion-button。
  • 標籤頁導航（Tabs）：.nav-tabs 搭配 .tab-content、.tab-pane 切換內容。
CSS / Bootstrap 類別
  • 導覽列顏色與位置：.bg-secondary、.fixed-top、.navbar-expand-sm。
  • 間距與排版：.mt-3、.pb-4、.p-5、.me-auto、.d-grid、.gap-2。
  • 顏色與文字：.bg-primary、.text-white、.btn-primary、.btn-success、.btn-dark。
  • 圖片：.w-100、.d-block、.img-fluid、.rounded、.rounded-3。
  • 提示框：.alert-success、.alert-danger、.alert-primary、.fade、.show。
  • 按鈕群：.btn-group、.btn-group-vertical、.active、.disabled。
  • 徽章定位：.position-relative、.position-absolute、.top-0、.start-100、.translate-middle。
  • 列表：.list-group、.list-group-item-action、.list-group-item-primary。
  • 分頁：.pagination、.page-item.disabled、.page-link.active。
  • Tabs：.nav-tabs、.nav-link.active、.tab-pane.active。
快捷鍵 / 工具
  • 快速建立 Bootstrap 網頁骨架：bs5（若有安裝相關 VS Code snippet 外掛 IntelliSense for CSS class）。
  • Emmet 快速產生結構：例如
    • nav.navbar>.container-fluid>a.navbar-brand
    • .carousel>.carousel-inner>.carousel-item*3
    • .card>img.card-img-top+.card-body>h4.card-title+p.card-text
    • ul.list-group>li.list-group-item*3
  • 透過多重選取（Alt+滑鼠 / Ctrl+D）一次修改多個 Title 或 Description。
  • 善用複製貼上同一組元件（例如卡片、按鈕群）再修改內容。

第十二堂課：Bootstrap 元件與排版應用

課堂練習 web12： Bootstrap 元件練習（Modal／進度條／表單）

HTML
  • 引用 Bootstrap 5 CDN（CSS＋JS）
  • 模態框（.modal、.modal-dialog、.modal-content）
  • 按鈕觸發 Modal：data-bs-toggle="modal"、data-bs-target="#modalId"
  • 邊框元素：<span class="border border-5 border-primary rounded-3 p-2">
  • 圖片浮動排版：class="float-start me-3"
  • 區塊置中與文字對齊：text-center、mx-auto、w-25、h-50
  • 進度條元件：.progress + .progress-bar
  • 表單結構：<form> 搭配 .row、.col-md-6、.form-control
CSS / Bootstrap 類別
  • border、border-5、border-primary、rounded-3：設定粗邊框與圓角
  • float-start、me-3：圖片左浮動與右邊外距
  • text-center、mx-auto：文字置中、區塊水平置中
  • progress、progress-bar：進度條元件
  • bg-danger、bg-success：進度條顏色
  • row、col-md-6、col-12：Bootstrap 栅格系統排版
  • form-control：統一 input、textarea 外觀
  • btn、btn-primary、btn-warning：按鈕樣式
快捷鍵 / 小重點
  • VS Code：Alt + Shift + ↓ / ↑ 複製一行
  • VS Code：Ctrl + / 加上或取消單行註解
  • Chrome：F12 開啟開發者工具，檢查元素與排版
  • Modal JS 初始化正確寫法：
    const myModal = new bootstrap.Modal(document.getElementById('modalId'));
    （注意是 Modal 不是 Model，ID 也要對到 modalId）
